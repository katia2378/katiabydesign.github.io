About the Project

An updated look for MACKAGE, a Montreal high end outwear fashion brand.

I created a cleaner aethtetic and improved user experience for this e-commerce website. I also updated the newsletter campaign with a mobile friendly template

The Challenge

The challenge was to make the salesforce cookie cutter template look on brand, while giving the end user a better a better shopping experience. There were many limitation on creativity, on what kind of design could be implemented within this box.

This project is continuously updated to enhance user UX and aethtetic

Goals

Update appearance and create a more user friendly experience

Design a responsive website with focus on mobile friendly interface
